# 🤝 Contributing to Awesome Developer Resources

Thank you for considering contributing to Awesome Developer Resources! 🎉

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button on the top right of this repository.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/awesome-developer-resources.git
cd awesome-developer-resources
```

### 3. Create a Branch

```bash
git checkout -b add-new-resource
```

### 4. Make Your Changes

- Add new resources to the appropriate section in `README.md`
- Follow the existing format and structure
- Keep descriptions clear and concise

### 5. Commit Your Changes

```bash
git add .
git commit -m "Add [Resource Name] - [Brief Description]"
```

### 6. Push and Create Pull Request

```bash
git push origin add-new-resource
```

Then open a Pull Request on GitHub!

## Contribution Guidelines

✅ **Do:**
- Add useful, high-quality resources
- Follow the existing format
- Include proper links and descriptions
- Categorize resources correctly

❌ **Don't:**
- Add duplicate resources
- Add low-quality or outdated resources
- Add resources without proper descriptions
- Spam or self-promote without adding value

## Resource Format

```markdown
### Category Name
| Name | Description | Link |
|------|-------------|------|
| Resource | Brief description | [Link](#) |
```

## Questions?

Feel free to open an issue if you have any questions!
